Note. If you are upgrading an existing installation of SPECCHIO V3.0.1 or earlier, ensure that you keep the file db_config.txt from your existing installation. This file contains your existing account configuration. To continue using these accounts with SPECCHIO V3.0.2, copy your db_config.txt file into the folder in which you installed SPECCHIO V3.0.2.
